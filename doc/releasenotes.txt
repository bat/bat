BAT - Bayesian Analysis Toolkit
------------------------------------------
Release notes for version:    0.9
Release date:                 2010-04-xx
Urgency:                      low
------------------------------------------

This version is intended to be a pre-release for the stable BAT version 1.0.
It contains many improvements and a few bugfixes. The most important changes
are summarized below. For detailed list of changes see the Changelog. For
more information consult the BAT webpage at http://www.mpp.mpg.de/bat/.

o The user interface for the models has changed. The vector of parameters
  is now passed as const reference which leads to significant speed up
  especially for models with large number of parameters.
  All user models have to be adjusted!!!

o Added new fully rewritten template fitter - BCMTF (MTF stands for
  MultiTemplateFitter). 
  ...
  The old BCTemplateFitter is kept for the moment but is depreceted.

o Fixed Makefile generation in CreateProject.sh script.

o Fixed compilation in the presence of more BAT versions in the system.

o Global code cleanup.

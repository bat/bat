Release notes for version:    0.2
Release date:                 2009-01-14
----------------------------------------

This version of BAT includes many updates. The major improvements are:

o  new installation procedure based on autoconf/automake
   - usage of Cuba is now optional but still recommended

o  prssibility to run BAT interactively from within ROOT

o  adding 'fast fitter' classes for graph fitting, histogram
   fitting and efficiency fitting allowing to do these basic types of
   fits with minimal effort

...

For more information consult the BAT webpage at http://www.mppmu.mpg.de/bat/
